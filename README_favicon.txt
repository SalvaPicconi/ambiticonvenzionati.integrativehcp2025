Salva qui l'immagine che vuoi usare come favicon con il nome: favicon.png

Formato consigliato: PNG quadrato (es. 512x512). In alternativa puoi usare SVG rinominandolo in favicon.svg e aggiornando il tag nel file index.html.

Attuale configurazione in index.html:
<link rel="icon" type="image/png" href="favicon.png">
<link rel="apple-touch-icon" href="favicon.png">
